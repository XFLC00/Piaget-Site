# Piaget Almada - Website de Acolhimento

Website estático para acolhimento de novos estudantes do Campus de Almada do Piaget.

## Conteúdo

- Página inicial com informação de acolhimento
- Guia para o primeiro dia no campus
- Mapa visual simplificado
- Serviços e cursos organizados por área
- Links úteis, FAQ e contactos
- Menu responsivo para telemóvel

## Como ver o site

Abre o ficheiro `index.html` num browser.

Também podes correr um servidor local simples dentro desta pasta:

```bash
python3 -m http.server 8000
```

Depois entra em:

```text
http://localhost:8000
```
